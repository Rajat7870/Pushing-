1. Git add --To add the file to the staging area
2.touch -- To make a black file in any format
3. GIT add -A  --To add all the file in the staging area 
4. GIT checkout --To reach the file to the previos version in case you don't want or match the file with previus commit
5. Git diff ---to check the difference between working directory and staging
6. Git -m -a ----Skip staging area and commit directly
7. Git rm ---Removing the file from working directory and staging area 
8. Git rm --cached  --- Remove the file from  staging area
9. Git status -S ----Showing the short status for Modified/ Commit
10. Git Ignore--Ignore the file
